# StockProject

## Overview
StockProject is a Django-based API application designed to manage and retrieve information about raw materials needed for product manufacturing in a production enterprise. The project allows users to check inventory levels, quantities of materials, and the relationship between products and their required materials.

## Features
- **Retrieve information about products and their corresponding raw materials.**
- **Manage warehouse inventory for tracking material availability.**
- **API endpoints for easy integration with other systems.**

## Technologies Used
- **Python**
- **Django**
- **Django Rest Framework (DRF)**
- **SQLite3**
- **Postman (for testing APIs)**
